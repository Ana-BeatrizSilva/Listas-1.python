## Sobre o Projeto

**Exercícios de Listas em Python** é uma coleção de atividades desenvolvidas com o objetivo de praticar e consolidar conhecimentos sobre manipulação de listas utilizando a linguagem Python.

Os exercícios abordam diferentes cenários encontrados no desenvolvimento de software, como processamento de dados, filtragem de informações, análise de métricas, controle de inventário, gerenciamento de carrinhos de compras, monitoramento de sensores e validação de dados.

Cada exercício foi implementado de forma independente, permitindo explorar operações fundamentais com listas, estruturas condicionais, laços de repetição, funções nativas da linguagem e interação com o usuário.

O projeto possui caráter educacional e foi desenvolvido como parte dos estudos em lógica de programação e fundamentos da linguagem Python.

## Objetivos de Aprendizagem

Este projeto foi criado para praticar e consolidar conhecimentos relacionados a:

- Criação e manipulação de listas;
- Percorrer listas utilizando laços de repetição;
- Filtragem de dados;
- Utilização de condicionais;
- Cálculo de médias, máximos e mínimos;
- Remoção de valores inválidos;
- Verificação de duplicatas;
- Ordenação de listas;
- Busca de elementos;
- Interação com o usuário através do terminal;
- Aplicação de lógica de programação em problemas reais.

---

## Exercícios Desenvolvidos

### 1. Aprovação de Alunos

- Análise de notas;
- Geração de lista contendo "Aprovado" ou "Reprovado";
- Uso de condicionais e listas.

### 2. Aplicação de Descontos

- Verificação de preços;
- Aplicação de desconto em produtos elegíveis;
- Filtragem de valores.

### 3. Carrinho de Compras

- Adição de itens;
- Remoção de itens;
- Exibição do conteúdo do carrinho;
- Utilização de `match` e `case`.

### 4. Monitoramento de Temperatura

- Identificação da maior temperatura;
- Identificação da menor temperatura;
- Cálculo da média das leituras.

### 5. Tratamento de Leituras de Sensor

- Remoção de dados inválidos;
- Validação de leituras;
- Cálculo da média das leituras válidas.

### 6. Inventário de Filamentos

- Cadastro de novas cores;
- Verificação de duplicatas;
- Ordenação alfabética dos dados.

### 7. Controle de Presença

- Remoção de nomes repetidos;
- Contagem de participantes únicos;
- Busca por nomes específicos.

### 8. Monitoramento de Tensão Elétrica

- Identificação de valores máximos e mínimos;
- Detecção de anomalias;
- Geração de alertas de manutenção.
